Setup Instructions
1. Unzip the file
2.

Change the database connections in config_db.php in Connections folder

$hostname_config_db = "localhost"; This is your hostname, most of the time it is localhost if different change it.
$database_config_db = "adesbook"; This is database name.
$username_config_db = "root"; Your username to login to your mySql database
$password_config_db = "dfjdlfj"; Your password to login to your mySql database
3.

Change the username and password for admin login page inin config_db.php in Connections folder

Default username and password is admin for username and admin for password. This is a login information for you Administrator Area. You will be able to manage your guestbook records; delete and modify.

4.

Create a table called adesbook with the following code:

CREATE TABLE adesbook (
ID int(8) NOT NULL auto_increment,
firstName varchar(30) NOT NULL default '',
lastName varchar(30) NOT NULL default '',
country varchar(30) default NULL,
email varchar(30) NOT NULL default '',
website varchar(200) default NULL,
cmnt text NOT NULL,
date varchar(100) NOT NULL default '',
marker date default '0000-00-00',
time timestamp(14) NOT NULL,
PRIMARY KEY (ID)
) TYPE=MyISAM;

Note: phpMyAdmin is a very useful tool to manage your databases, you can download it free online at http://www.phpmyadmin.net
5. Change the email to your own in send_mail.php - notification of new message entry is sent to this email.
6. Upload all the files and folders to your server.
7. You are done!

Viewing your Guestbook
http://www.yourdomain.com/AdesGuestbook/read.php is the url for the Read Messages section
http://www.yourdomain.com/AdesGuestbook/admin/index.php is the url for the Admin Section

Customizing your Guestbook
Change forText.css to modify the look of the Guestbook to suit your website.
Change the header.php and footer.php to encapsulate the Guestbook inside your website.
