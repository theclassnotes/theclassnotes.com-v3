AdesGuestbook 2.1
AdesGuestbook works with PHP and MySql. Admin Page is a secure page that can be logged in only by admininstrator and includes functions such as deleting the record and modifying the record. It is developed with the customization in mind, so you can easily change the look of the AdesGuestbook according to your website. It uses one single CSS file for the table colors and text format. By changing this CSS file you can apply your choice of colors easily to the whole Guestbook.

Full Features:

  • Captcha has been integrated into sign page
  • Validates the user message when signing the guestbook
    • Validates Name - not to be empty
    • Validates Email - Email is validated on email@website.com format
    • Validates Message - not to be emtpy
    • When optional fields are left empty, these fields are not displayed on the viewing page. This will prevent guestbook from showing unnecessary emtpy table rows.
  • Email notification is sent to the admin when a new message is posted.
  • Look & Feel of the guestbook is controlled by a central CSS file.
  • Easy to integrate to any website.
  • Includes a secure Admin section.
  • Administrator can modify and delete the messages.
  • Administrator can delete the message based on: Name and Email.
